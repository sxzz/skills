# Kevin Deng's Skills

A curated collection of [Agent Skills](https://agentskills.io/home) reflecting [Kevin Deng](https://github.com/sxzz)'s preferences, experience, and best practices for TypeScript projects.

## Installation

```bash
pnpx skills add sxzz/skills --skill='*'
```

or to install all of them globally:

```bash
pnpx skills add sxzz/skills --skill='*' -g
```

Learn more about the CLI usage at [skills](https://github.com/vercel-labs/skills).

## Skills

| Skill               | Description                                                                                |
| ------------------- | ------------------------------------------------------------------------------------------ |
| [sxzz](skills/sxzz) | Kevin Deng's preferences and best practices for TypeScript projects (eslint, tsdown, etc.) |
